\section{Disclosure of Use of LLMs (ICLR 2026 Requirement)}
In line with the \href{https://blog.iclr.cc/2025/08/26/policies-on-large-language-model-usage-at-iclr-2026/}{LLM usage disclosure policy} for ICLR 2026 submissions, we report our usage of LLMs as the following:
\begin{itemize}
    \item Design and polish matplotlib and seaborn figures in the paper (ChatGPT)
    \item Write LaTeX code for tables, figures, and listings, including aesthetically enhancing the styles (ChatGPT)
    \item Polish and edit text in the paper (ChatGPT)
    \item Find relevant citations for related work (ChatGPT)
    \item Assist in producing code for experiments (GitHub Copilot in VSCode, ChatGPT)
\end{itemize}
